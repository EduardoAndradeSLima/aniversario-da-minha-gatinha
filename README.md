Meu Primeiro Projeto

Este é o primeiro projeto que desenvolvo, feito com muito carinho para criar uma experiência especial de aniversário. Ele reúne um bolo animado em CSS, uma carta com efeito de digitação, uma pequena galeria de fotos e uma surpresa final. Tudo feito apenas com **HTML, CSS e JavaScript**, sem frameworks.

A ideia foi aprender enquanto criava algo significativo: animações, interações, efeitos visuais e organização do fluxo da página. Mesmo sendo meu primeiro projeto, busquei deixá‑lo bonito, leve e cheio de emoção.

✔ Bolo animado em CSS
✔ Carta digitada automaticamente
✔ Galeria simples e responsiva
✔ Botão divertido que "foge"
✔ Final com mensagem especial

Foi meu ponto de partida no mundo da programação e também uma forma de transformar código em sentimento.
